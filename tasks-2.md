> –ü–æ—è—Å–Ω–µ–Ω–∏–µ: —Å—Ç—Ä–æ–∫–∏ s –∏ t –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è –∏–∑–æ–º–æ—Ä—Ñ–Ω—ã–º–∏, –µ—Å–ª–∏ –≤—Å–µ –≤—Ö–æ–∂–¥–µ–Ω–∏—è –∫–∞–∂–¥–æ–≥–æ —Å–∏–º–≤–æ–ª–∞ —Å—Ç—Ä–æ–∫–∏ s –º–æ–∂–Ω–æ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ –∑–∞–º–µ–Ω–∏—Ç—å –¥—Ä—É–≥–∏–º —Å–∏–º–≤–æ–ª–æ–º –∏ –ø–æ–ª—É—á–∏—Ç—å —Å—Ç—Ä–æ–∫—É t. –ü–æ—Ä—è–¥–æ–∫ —Å–∏–º–≤–æ–ª–æ–≤ –ø—Ä–∏ —ç—Ç–æ–º –¥–æ–ª–∂–µ–Ω —Å–æ—Ö—Ä–∞–Ω—è—Ç—å—Å—è, –∞ –∑–∞–º–µ–Ω–∞ ‚Äî –±—ã—Ç—å —É–Ω–∏–∫–∞–ª—å–Ω–æ–π. –¢–∞–∫, –¥–≤–∞ —Ä–∞–∑–Ω—ã—Ö —Å–∏–º–≤–æ–ª–∞ —Å—Ç—Ä–æ–∫–∏ s –Ω–µ–ª—å–∑—è –∑–∞–º–µ–Ω–∏—Ç—å –æ–¥–Ω–∏–º –∏ —Ç–µ–º –∂–µ —Å–∏–º–≤–æ–ª–æ–º –∏–∑ —Å—Ç—Ä–æ–∫–∏ t, –∞ –≤–æ—Ç –æ–¥–∏–Ω–∞–∫–æ–≤—ã–µ —Å–∏–º–≤–æ–ª—ã –≤ —Å—Ç—Ä–æ–∫–µ s –¥–æ–ª–∂–Ω—ã –∑–∞–º–µ–Ω—è—Ç—å—Å—è –æ–¥–Ω–∏–º –∏ —Ç–µ–º –∂–µ —Å–∏–º–≤–æ–ª–æ–º.

### –†–µ–∞–ª–∏–∑–∞—Ü–∏—è

```python
def is_isomorphic(s: str, t: str) -> bool:
    if len(s) != len(t):
        return False
    
    # –°–ª–æ–≤–∞—Ä—å: –∫–∞–∫–æ–π —Å–∏–º–≤–æ–ª –∏–∑ s —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç —Å–∏–º–≤–æ–ª—É –≤ t
    s_to_t = {}
    # –°–ª–æ–≤–∞—Ä—å: –∫–∞–∫–æ–π —Å–∏–º–≤–æ–ª –∏–∑ t –ø—Ä–∏—à—ë–ª –æ—Ç —Å–∏–º–≤–æ–ª–∞ –≤ s
    t_to_s = {}
    
    for a, b in zip(s, t):
        # –ï—Å–ª–∏ —Å–∏–º–≤–æ–ª –∏–∑ s —É–∂–µ –≤—Å—Ç—Ä–µ—á–∞–ª—Å—è
        if a in s_to_t:
            if s_to_t[a] != b:
                return False  # –ù–æ –æ–Ω –¥–æ–ª–∂–µ–Ω –±—ã–ª —Å—Ç–∞—Ç—å –¥—Ä—É–≥–∏–º ‚Äî –æ—à–∏–±–∫–∞!
        # –ï—Å–ª–∏ —Å–∏–º–≤–æ–ª –∏–∑ t —É–∂–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª—Å—è, –Ω–æ –Ω–µ –¥–ª—è a
        elif b in t_to_s:
            if t_to_s[b] != a:
                return False  # –≠—Ç–æ—Ç —Å–∏–º–≤–æ–ª –≤ t —É–∂–µ "–∑–∞–Ω—è—Ç" –¥—Ä—É–≥–∏–º
        # –û–±–∞ —Å–∏–º–≤–æ–ª–∞ –Ω–æ–≤—ã–µ ‚Äî —Ñ–∏–∫—Å–∏—Ä—É–µ–º —Å–≤—è–∑—å
        else:
            s_to_t[a] = b
            t_to_s[b] = a

    return True
```

### ‚úÖ –ü—Ä–∏–º–µ—Ä—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è
